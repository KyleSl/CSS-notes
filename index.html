CSS Code Cheatsheet

Flex Box

Flex boxes are a very useful way to format information. They are used to make sure that every element fits within a certain container. On this website, all of these cards are within one big flex container, which allows them to scale. Using media queries, seen on another card, we are able to make sure that the page doesn't break on smaller screens. 


.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

Grid

Grids are another way to format information. I am not super sure how they work but I will update this in the future. 


.grid {
    display: grid;
}

Media Queries

Media queries are used to get information about the device that the website is displaying on. The most common use is to find the screen size and change certain elements accordingly. For example, this website aligns everything into one column. 


@media screen and (max-width: 768px) {
    .flex-container {
        flex-direction: column;
    }
}

Element Selectors

Element selectors are used in CSS to pick the specific elements you want to style. If you want to be good with CSS, you need to master these selectors. 


header {}
header h1 {}
#navbar ul li {}
.container ul li + li {}

Classes & IDs

Classes and IDs are used in HTML and CSS to help with the element selection process. You can set a class or ID to an element in HTML and modify it in CSS. This is another thing you have to master if you want to be fluent in CSS. IDs take priority over classes. 


section class="card"

.card {
    font-size: 200%;
}
